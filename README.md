# To-Do List com Autenticação (Flask)

Projeto desenvolvido em Python utilizando Flask com sistema de gerenciamento de tarefas.

## Funcionalidades
- Criação de tarefas
- Exclusão e conclusão das tarefas
- Listagem de tarefas
- Organização de rotas e templates

## Tecnologias
- Python
- Flask
- HTML/CSS
- SQLite

## Objetivo
Projeto criado para praticar conceitos de back-end e front-end, rotas e estruturação de aplicações web.
